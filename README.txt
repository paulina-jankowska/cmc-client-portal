GITHUB PAGES
1. Create a public GitHub repository.
2. Upload all files and the documents folder.
3. Open Settings -> Pages.
4. Select Deploy from a branch, main, /root.
5. Save.

Notes:
- The site includes PL/EN/DE/CS/SK interface switching.
- Czech and Slovak agreements are opened locally from the documents folder.
- Cooperation plans are user-supplied service levels. Their final wording and legal applicability should match individual client agreements.
